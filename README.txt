O site tem como objetivo a avaliação para possíveis processos seletivos na área do desenvolvimento web
Foram utilizadas as tecnologias HTML5, CSS3, Javascript, Bootstrap e Owl Carousel para a confecção do site.
Na maioria do site foi utilizado apenas o HTML, CSS e Bootstrap, 
sendo que o primeiro slider é nativo do Bootstrap e o segundo (com os produtos) foi utilizado o framework owl carousel.
Todos os arquivos estão organizados em uma estrutura de pastas, a fim de manter a organização e integridade do código-fonte.
Site realizado por Fernando Almeida, e feito o upload no GitHub pages no dia 28/02/2021.
